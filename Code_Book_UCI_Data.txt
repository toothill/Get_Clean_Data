UCI Data Code Book - UCI Data

Subject - Subject
	Description:  Number for the participant of the data collection
	Type:  Integer
	Values:  1-30
	
Activity_Nbr - Activity Number 
	Description:  Code corresponding to Activity Label below
	Type:  Integer
	Values:  1-6
	
Activity_Label - Activity Label	
	Description:  Descriptive label for each activity 1-6	
	Type:  Character
	Values:  WALKING (1)
			 WALKING_UPSTAIRS (2)
			 WALKING_DOWNSTAIRS (3)
			 SITTING (4)
			 STANDING (5)
			 LAYING (6)
